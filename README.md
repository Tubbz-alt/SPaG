# Grammar-Generator

[![LICENSE](https://img.shields.io/badge/LICENSE-MIT-green.svg)](https://github.com/rrozansk/Grammar-Generator/blob/master/LICENSE.txt) [![RELEASES](https://img.shields.io/badge/Releases-current-yellow.svg)]()

## Prerequisites
### Generator:
  - python 2.7

### Generated code:
  - gcc (Ubuntu/Linaro 4.6.3-1ubuntu5) 4.6.3 or equivalent C compiler

## Getting Started
```sh
$ git clone https://github.com/rrozansk/Grammar-Generator.git
$ cd Grammar_Generator
$ vim main.py # edit the grammar to needs/wants, and what to generate
$ ./main      # produce a file with generated code or error
```
